Pipeline Control

Trigger the full analytics pipeline for one ISO or all ISOs. This re-runs ingest → transform → forecast → queue → markets → finance → charts and refreshes all dashboard data.

REST API

Access any ISO's data programmatically via the JSON API:

GET /api/isos GET /api/{iso}/panel → timestamps, load_mw, price_usd_mwh, temperature_f GET /api/{iso}/market → avg price, solar/wind capture, negative hours GET /api/{iso}/queue → queue outlook by year and technology GET /api/{iso}/forecast → load forecast scenarios (low / base / high) GET /api/{iso}/finance → scenario matrix, base summary, sensitivity POST /api/run/{iso} → trigger full pipeline for one ISO POST /api/run-all → trigger all ISOs
