Pipeline Control

Trigger the full analytics pipeline for one ISO or all ISOs. Runs: ingest → transform → forecast → queue → markets → finance → charts. Dashboard data refreshes automatically.

REST API Reference

Access all data programmatically via the JSON API:

GET /api/isos GET /api/{iso}/panel → timestamps, load_mw, price_usd_mwh, temperature_f GET /api/{iso}/market → KPI metrics dict GET /api/{iso}/hourly → enriched hourly: solar/wind profile, congestion GET /api/{iso}/price-duration → price duration curve (percentile → price) GET /api/{iso}/monthly → monthly aggregations: avg price, load, neg hours GET /api/{iso}/heatmap → hour-of-day × day price/load matrix GET /api/{iso}/queue → queue outlook by year and technology GET /api/{iso}/queue-detail → individual project list with status breakdown GET /api/{iso}/forecast → load scenarios (low/base/high) by year GET /api/{iso}/backtest → forecast backtest actual vs predicted GET /api/{iso}/finance → scenario matrix, base summary, sensitivity GET /api/{iso}/cashflows → year-by-year projected cash flows GET /api/{iso}/findings → market findings text POST /api/run/{iso} → trigger full pipeline for one ISO POST /api/run-all → trigger all ISOs
